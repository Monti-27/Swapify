# Step-by-Step Tutorials

In this section, you will learn how to set up your first Boomerang trade using the WeSwap Dashboard.

## 1. Setting Up a Boomerang Trade

* **Step 1: Select Mode**
    Click on the **"Boomerang Mode"** tab from the main menu on the Dashboard.
* **Step 2: Select Asset (WSP Coin)**
    Choose the main asset you will use to enter the trade (Usually `WSP` or `USDC`).
    * *Example: I want to start the trade with 5 SOL from my wallet.*
* **Step 3: Define Target (Target Coin)**
    Paste the **Contract Address (CA)** of the token you wish to buy into the input box. WeSwap will automatically fetch the token details.
* **Step 4: Strategy Settings**
    * **Amount:** Enter the amount you wish to invest.
    * **Take Profit (%):** Set the percentage gain at which the system should sell the target coin (e.g., 50%).
    * **Stop Loss (%):** (Optional) Set the level at which you want to cut your losses.
* **Step 5: Start**
    Review your settings and click the **"Start Boomerang"** button. The automation begins once you approve the wallet transaction.

## 2. Monitoring and Managing Trades

Your initiated trades are listed in the **"Active Trades"** panel.
* **Live Status:** You can monitor your Profit/Loss (PnL) in real-time.
* **Manual Override:** If you wish to end the trade without waiting for the automation, you can use the **"Force Sell"** or **"Cancel"** buttons.
